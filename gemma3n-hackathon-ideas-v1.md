# Gemma3n 해커톤 아이디어 브레인스토밍 V1 - 초기 10개 핵심 아이디어

## 🎯 아이디어 선정 기준
- **실제 문제 해결**: 명확하고 시급한 실제 문제
- **오프라인 필수성**: 인터넷 없이 작동해야 하는 명확한 이유
- **멀티모달 활용**: 이미지, 오디오, 텍스트를 자연스럽게 활용
- **3분 데모 임팩트**: 시각적으로 강력한 Before/After
- **2주 구현 가능**: MVP 수준으로 실제 작동 가능

## 🏥 의료/건강 분야

### 1. 응급 의료 시각 보조 (Emergency Medical Visual Assistant)
**문제**: 시골 보건소 직원이 의사 없이 피부질환/상처를 진단할 수 없음
**해결**: 
```
[시골 보건 직원]이 [인터넷 없는 오지]에서 겪는 [환자 진단 불가]를
[이미지 분석 + 의료 RAG]를 활용해 [즉석 진단 및 처치 가이드]로 해결
```
**구현**:
- MediaPipe로 상처/피부 영역 감지
- 사전 학습된 피부질환 분류 모델
- 오프라인 의료 가이드라인 RAG
- 현지어 음성 출력
- Mix'n'Match: UI(2B) + 진단(4B)

**데모 시나리오**:
- 0:30 - 오지 마을, 아이의 발진을 보고 당황하는 보건 직원
- 1:00 - 앱으로 촬영, 즉시 "농가진" 진단 + 처치법 안내
- 0:30 - 치료 후 호전된 아이의 모습

---

### 2. 노인 약물 관리 도우미 (Elderly Medication Guardian)
**문제**: 독거노인이 여러 약물 복용 시 혼동하여 위험
**해결**:
```
[독거노인]이 [매일 아침저녁]에 겪는 [복잡한 약물 구분]을
[이미지 인식 + 음성 안내]를 활용해 [시각/청각 복약 가이드]로 해결
```
**구현**:
- YOLO 기반 알약 형태/색상 인식
- 처방전 OCR → 복약 스케줄 생성
- 개인화된 음성 알림 (자녀 목소리 합성)
- 복약 이력 자동 기록
- 위험 상황 시 가족 알림

**데모 시나리오**:
- 0:30 - 약통 앞에서 혼란스러워하는 할머니
- 1:00 - 카메라로 약 촬영 → "혈압약입니다. 아침 식후 1알" 음성 안내
- 0:30 - 안심하고 정확히 복용하는 모습

## 🌾 농업/환경 분야

### 3. 작물 질병 현장 스캐너 (Plant Disease Field Scanner)
**문제**: 개발도상국 소농이 작물 질병으로 수확 실패
**해결**:
```
[소규모 농부]가 [전문가 없는 농촌]에서 겪는 [작물 질병 진단]을
[이미지 분석 + 처방 RAG]를 활용해 [즉석 진단 및 치료법]으로 해결
```
**구현**:
- PlantNet 스타일 질병 분류기
- 지역별 병충해 데이터베이스
- 현지어 음성 설명
- 유기농 처방 우선 제공
- GPS 기반 지역 특화 조언

**데모 시나리오**:
- 0:30 - 시들어가는 토마토밭을 보며 걱정하는 농부
- 1:00 - 잎 촬영 → "역병입니다. 구리 용액 살포하세요" + 제조법
- 0:30 - 처방 후 되살아난 작물

## ♿ 접근성/포용 분야

### 4. 수어 긴급 통역기 (Sign Language Bridge)
**문제**: 청각장애인이 응급상황에서 의사소통 불가
**해결**:
```
[청각장애인]이 [응급실/경찰서]에서 겪는 [의사소통 장벽]을
[수어 인식 + TTS]를 활용해 [실시간 양방향 통역]으로 해결
```
**구현**:
- MediaPipe Holistic으로 수어 동작 추적
- 수어 → 텍스트 변환 모델
- 텍스트 → 음성 출력
- 음성 → 텍스트 → 수어 애니메이션
- 긴급 상황 퀵 버튼

**데모 시나리오**:
- 0:30 - 응급실에서 수어로 필사적으로 설명하는 환자
- 1:00 - 앱이 "배가 아파요" 음성 변환, 의사 말을 텍스트로
- 0:30 - 원활한 진료를 받는 모습

### 5. 시각장애인 실내 네비게이터 (Accessibility Navigation Assistant)
**문제**: 시각장애인이 처음 가는 건물 내부 이동 어려움
**해결**:
```
[시각장애인]이 [낯선 건물]에서 겪는 [실내 길찾기]를
[객체 인식 + 공간 오디오]를 활용해 [음성 공간 안내]로 해결
```
**구현**:
- YOLO로 문, 계단, 엘리베이터 감지
- 깊이 추정으로 거리 계산
- 3D 공간 음향으로 방향 안내
- 장애물 경고음
- 점자 블록 인식

**데모 시나리오**:
- 0:30 - 병원 로비에서 진료실을 찾는 시각장애인
- 1:00 - "우측 10미터 엘리베이터" 공간 음향 안내
- 0:30 - 목적지 도착 확인

## 🌍 다문화/난민 지원

### 6. 난민 서류 도우미 (Refugee Documentation Helper)
**문제**: 난민이 국경에서 언어 장벽으로 서류 작성 불가
**해결**:
```
[난민]이 [국경 검문소]에서 겪는 [서류 작성 어려움]을
[OCR + 번역 + 음성]을 활용해 [모국어 작성 지원]으로 해결
```
**구현**:
- 다국어 OCR (20개 언어)
- 양식 필드 자동 감지
- 음성으로 모국어 입력
- 자동 번역 후 양식 기입
- 법적 용어 설명

**데모 시나리오**:
- 0:30 - 영어 서류 앞에서 당황한 시리아 난민 가족
- 1:00 - 아랍어로 말하면 자동 번역/기입
- 0:30 - 완성된 서류로 입국 승인

### 7. 토착어 보존 도구 (Indigenous Language Preserver)
**문제**: 소수민족 언어가 기록 없이 소멸 위기
**해결**:
```
[원주민 공동체]가 [현대화 과정]에서 겪는 [언어 소멸]을
[음성 인식 + 시각 사전]을 활용해 [인터랙티브 보존]으로 해결
```
**구현**:
- 음성 녹음 및 음소 분석
- 사물 촬영 → 단어 매칭
- 스토리텔링 녹화
- 어린이 학습 게임
- 가족 간 전승 도구

**데모 시나리오**:
- 0:30 - 손자와 말이 통하지 않는 원주민 할머니
- 1:00 - 할머니 목소리로 언어 학습 앱 생성
- 0:30 - 손자가 할머니 언어로 대화

## 🚨 재난/안전 분야

### 8. 재난 생존자 탐지기 (Disaster Survivor Locator)
**문제**: 지진 잔해에서 생존자 발견 어려움
**해결**:
```
[구조대]가 [재난 현장]에서 겪는 [생존자 탐지]를
[음향 분석 + 열화상]을 활용해 [정확한 위치 파악]으로 해결
```
**구현**:
- 음향 패턴 인식 (신음, 두드림)
- 열화상 이미지 분석
- 3D 음향으로 방향 표시
- 팀원 간 오프라인 공유
- 구조 우선순위 제안

**데모 시나리오**:
- 0:30 - 무너진 건물에서 수색하는 구조대
- 1:00 - 미세한 두드림 소리 감지 → 위치 표시
- 0:30 - 생존자 구조 성공

### 9. 건설 현장 안전 감시자 (Construction Safety Inspector)
**문제**: 건설 현장 안전 수칙 위반으로 사고 발생
**해결**:
```
[건설 노동자]가 [작업 현장]에서 겪는 [안전 사고 위험]을
[실시간 영상 분석]을 활용해 [즉각 경고 시스템]으로 해결
```
**구현**:
- 안전모/안전띠 착용 감지
- 위험 구역 진입 경고
- 추락 위험 상황 인식
- 다국어 음성 경고
- 사고 자동 기록

**데모 시나리오**:
- 0:30 - 안전모 없이 작업하는 인부
- 1:00 - "안전모 착용!" 경고 + 관리자 알림
- 0:30 - 사고 예방된 현장

### 10. 식품 알레르기 보호자 (Food Allergy Guardian)
**문제**: 심각한 알레르기 환자가 해외여행 중 위험
**해결**:
```
[알레르기 환자]가 [해외 식당]에서 겪는 [성분 확인 불가]를
[OCR + 성분 분석]을 활용해 [즉석 알레르기 경고]로 해결
```
**구현**:
- 메뉴/포장 텍스트 OCR
- 성분명 다국어 매칭
- 알레르기 유발 성분 DB
- 위험도 시각/음성 경고
- 대체 메뉴 제안

**데모 시나리오**:
- 0:30 - 태국 식당에서 메뉴 보는 땅콩 알레르기 환자
- 1:00 - 메뉴 스캔 → "땅콩 소스 포함!" 경고
- 0:30 - 안전한 다른 메뉴 주문
