프로젝트 개요

최첨단 온디바이스 AI를 활용하여 디지털 소외 계층의 키오스크 접근성을 혁신하는 프로젝트입니다.

프로젝트 비전 및 목표

본 프로젝트의 비전은 Google의 Gemma 3n 모델을 탑재한 온디바이스 AI 에이전트를 통해 시각 장애인 및 고령층이 타인의 도움 없이 키오스크를 완벽하게 사용할 수 있도록 지원하는 것입니다. 사용자의 음성 명령과 스마트폰 카메라를 통해 키오스크 화면을 실시간으로 인지하고, 사용자를 대신하여 UI를 조작하는 지능형 에이전트를 개발하여, 모든 사람이 동등하게 디지털 사회의 편의를 누리는 환경을 조성하는 것이 궁극적인 목표입니다.

기술 스택

프로젝트를 구성하는 핵심 기술 요소와 아키텍처를 소개합니다.

핵심 엔진: Gemma 3n

Gemma 3n은 단순한 경량 모델이 아닌, 엣지 환경에서의 효율적 구동을 위해 설계된 근본적인 아키텍처 혁신을 담고 있어 본 프로젝트에 최적입니다.

MatFormer 아키텍처

하나의 모델 안에 여러 하위 모델이 중첩되어, 상황에 따라 연산량을 동적으로 조절하며 배터리 효율을 극대화합니다.

PLE & KV 캐싱

VRAM 요구사항을 줄이고, 초기 응답 속도를 2배 향상시켜 실시간 상호작용을 가능하게 합니다.

고성능 멀티모달

MobileNet-V5(Vision)와 USM(Audio) 인코더로 키오스크 화면과 사용자 음성을 빠르고 정확하게 인식합니다.

안드로이드 & 모델 배포 스택

최신 안드로이드 기술과 효율적인 모델 파인튜닝 및 배포 파이프라인을 통해 아이디어를 현실로 만듭니다.

  • 언어 및 UI: Kotlin & Jetpack Compose로 빠르고 현대적인 앱 개발
  • 카메라: CameraX로 안정적인 실시간 비디오 프레임 확보
  • 추론 엔진: MediaPipe LLM Inference API로 Gemma 3n 온디바이스 구동
  • 모델 튜닝: Unsloth를 사용하여 Colab 무료 GPU 환경에서 QLoRA 파인튜닝
  • 모델 배포: AI Edge Torch로 PyTorch 모델을 TFLite/.task 번들로 변환하여 앱에 탑재

사용자 경험(UX)

기술을 사용자 관점에서 구체화하여, 앱이 실제로 어떻게 동작하고 어떤 가치를 제공하는지 생생하게 보여줍니다.

인터랙티브 시나리오: 카페 키오스크 주문

시각 장애가 있는 사용자가 키오스크에서 음료를 주문하는 여정을 시뮬레이션합니다. '다음 단계' 버튼을 눌러 과정을 진행해보세요.

'신뢰할 수 있는 전문가' UX 프레임워크

단순한 음성 제어를 넘어, 사용자가 믿고 의지할 수 있는 '디지털 비서'를 만들기 위해 다음과 같은 대화 원칙을 적용합니다.

✔️

명시적 확인

"5,000원을 결제합니다. 맞습니까?" 와 같이 금전적 지출 전 반드시 사용자의 동의를 구합니다.

❓

능동적 명료화

"늘 드시던 것은 '라지 아이스 아메리카노'가 맞습니까?" 처럼 모호한 명령은 되물어 명확히 합니다.

⚠️

정중한 오류 처리

"죄송합니다. 작업에 실패했습니다. 다시 시도할까요?" 와 같이 실패 상황을 명확히 보고하고 대안을 제시합니다.

모델 학습 전략

범용 Gemma 3n 모델을 키오스크 제어에 특화된 전문가 모델로 탈바꿈시키기 위한 데이터 전략과 파인튜닝 계획입니다.

UI 제어를 위한 지도 파인튜닝(SFT)

모델이 UI를 제어하는 '언어'를 학습하도록, '명령어-행동' 쌍으로 구성된 고품질 마이크로 데이터셋(50~100개)을 구축합니다.

행동 공간(Action Space) JSON 스키마 예시:

{
  "action": "CLICK",
  "reasoning": "사용자가 커피를 요청했음.",
  "target_element_description": "'커피' 텍스트 버튼",
  "coordinates": [430, 250]
}

Unsloth를 활용한 QLoRA 튜닝

메모리 효율적인 QLoRA 파인튜닝을 Unsloth 라이브러리를 통해 2배 더 빠르게 수행합니다. 이를 통해 Colab의 무료 T4 GPU만으로도 80억 파라미터 모델의 튜닝이 가능합니다.
